# Change Log

All notable changes to the "fastled-wasm" extension will be documented in this file.

## [1.0.0] - 2024-01-01

### Added
- Initial release of FastLED WASM VS Code Extension
- Compile & Run commands with multiple build modes (quick, debug, release)
- Project initialization with FastLED examples
- Compiler server management (start/stop)
- Syntax highlighting for Arduino (.ino) files with FastLED-specific keywords
- Code snippets for common FastLED patterns
- Context menus for easy access to compilation commands
- Configuration settings for default behavior
- Automatic project detection for FastLED sketches
- Browser integration for viewing compiled output
- Update and maintenance commands for the FastLED compiler
- Comprehensive documentation and usage examples

### Features
- **Compilation**: One-click compilation with automatic browser opening
- **Project Management**: Initialize new projects with example code
- **Syntax Support**: Full syntax highlighting and code completion
- **Snippets**: Common patterns and boilerplate code
- **Server Management**: Local compiler server control
- **Configuration**: Customizable settings for different workflows
- **Integration**: Seamless integration with FastLED WASM compiler CLI
